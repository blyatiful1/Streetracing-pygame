Diese Python-Module müssen installiert sein, damit das Spiel läuft:
- pygame-ce
- pygame-menu
- pytmx

Der nachfolgende Befehl installiert alle Module:
pip install pygame pygame-ce pygame-menu pytmx

Notizen:
Ändern der Auflösung verändert nicht die größe des menüs
Highscoreliste ab zu vielen highscores unlesbar
